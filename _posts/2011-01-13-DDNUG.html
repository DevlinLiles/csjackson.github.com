Rapid Application Development for Silverlight Business Applications

Further R&D: Developmentor Sneed’s blog is there, which I now follow. + webinars.

Terms I needed to look up:

WCF: Windows Communication Foundation: a system that allows for asynchronous communication. A simple example is a chat program that isn’t constantly sending messages, but is capable of it whenever either side needs to.

RIA: Rich Internet Application: an app that is designed to run in a browser window. Flash games come to mind.
